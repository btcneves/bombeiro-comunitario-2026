# Figuras e Vícios de Linguagem

## 💡 Conceito
**Figuras**: recursos expressivos que enriquecem o texto literário.
**Vícios**: desvios da norma culta que prejudicam a clareza ou a correção.

## ⚡ Figuras mais cobradas

| Figura | Definição | Exemplo |
|--------|-----------|---------|
| **Metáfora** | Comparação implícita sem "como" | "O bombeiro é um leão no combate." |
| **Comparação (Símile)** | Comparação explícita com "como", "tal qual" | "O bombeiro é **como** um leão." |
| **Metonímia** | Substituição por relação lógica (parte/todo, causa/efeito) | "Li Machado" (autor/obra); "beber um copo" (conteúdo/recipiente) |
| **Hipérbole** | Exagero expressivo | "Já te disse mil vezes!" |
| **Eufemismo** | Suavizar algo desagradável | "Ele passou desta para melhor" (morreu) |
| **Ironia** | Sentido oposto ao literal | "Que pontualidade!" (para quem chegou atrasado) |
| **Antítese** | Oposição de ideias | "Unidos na dor, separados na alegria." |
| **Paradoxo** | Contradição aparente com verdade profunda | "Quanto mais aprendo, mais não sei." |
| **Personificação (prosopopeia)** | Atribuir características humanas a seres inanimados | "O vento sussurrou meu nome." |
| **Pleonasmo literário** | Redundância expressiva/intencional | "Chorar lágrimas de sangue" |
| **Sinestesia** | Mistura de sentidos (audição + visão etc.) | "Voz morna e aveludada" |
| **Anáfora** | Repetição no início de frases | "Virá o dia, virá a luz, virá a paz." |
| **Eufonia** | Som harmonioso | Aliteração, assonância |

## ⚡ Vícios de Linguagem

| Vício | Definição | Exemplo |
|-------|-----------|---------|
| **Pleonasmo vicioso** | Redundância desnecessária | "subir para cima", "entrar para dentro", "vi com meus próprios olhos" |
| **Ambiguidade (anfibologia)** | Duplo sentido não intencional | "Vi o bombeiro com o binóculo" (quem tinha o binóculo?) |
| **Cacofonia** | Som desagradável na junção de palavras | "a cada hora" → "acada hora" |
| **Barbarismo** | Erro de pronúncia, grafia, forma | "namorado" → "namoraido" |
| **Solecismo** | Erro sintático | "A gente fomos" (concordância errada) |
| **Estrangeirismo desnecessário** | Uso de palavra estrangeira com equivalente em português | "delete" em vez de "apague" |

## 🎯 Pegadinhas
- Pleonasmo **literário** (aceitável) ≠ pleonasmo **vicioso** (erro)
- "Surpresa inesperada" = pleonasmo vicioso (toda surpresa é inesperada)
- Metáfora sem "como" ≠ comparação com "como"
- Metonímia: autor por obra ("li Drummond"), parte por todo, continente por conteúdo

## 🃏 Flashcards
#flashcards

Qual é a diferença entre metáfora e comparação (símile)?::Metáfora = comparação implícita, sem "como". Comparação (símile) = explícita, com "como", "tal qual". Ex: "ele é um leão" (metáfora) vs "ele é como um leão" (comparação).

O que é metonímia?::Substituição de uma palavra por outra com relação lógica: causa/efeito, autor/obra, parte/todo, conteúdo/recipiente. Ex: "Ler Machado" (autor pela obra).

Cite 3 exemplos de pleonasmo vicioso::Subir para cima, entrar para dentro, surpresa inesperada, hemorragia de sangue, conviver junto.

"Quanto mais aprendo, mais não sei" — qual figura de linguagem?::Paradoxo (contradição aparente que contém uma verdade mais profunda).

O que é ambiguidade (anfibologia)?::Duplo sentido não intencional numa frase. Ex: "Vi o médico com o telescópio" — quem tinha o telescópio?
