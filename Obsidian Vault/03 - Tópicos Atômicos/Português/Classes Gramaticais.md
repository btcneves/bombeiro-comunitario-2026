# Classes Gramaticais

## 💡 Conceito
As 10 classes de palavras da língua portuguesa, classificadas por forma, função e significado.

## ⚡ Resumo Brutal

| Classe | Função | Flexiona? |
|--------|--------|-----------|
| **Substantivo** | nomeia seres, coisas, sentimentos | gênero, número, grau |
| **Artigo** | determina/indefine o substantivo | gênero, número |
| **Adjetivo** | caracteriza o substantivo | gênero, número, grau |
| **Numeral** | quantidade ou ordem | gênero, número (alguns) |
| **Pronome** | substitui ou acompanha nome | gênero, número, pessoa |
| **Verbo** | ação, estado, fenômeno | modo, tempo, número, pessoa |
| **Advérbio** | modifica verbo, adj. ou outro adv. | grau (mais/menos) |
| **Preposição** | liga termos, indica relação | invariável |
| **Conjunção** | liga orações ou palavras | invariável |
| **Interjeição** | expressa emoção | invariável |

## 🎯 Pegadinhas — Transitividade e Polissemia
- "Bastante" pode ser adjetivo, pronome indefinido ou advérbio → identificar pelo contexto
- "Muito" como advérbio é invariável; como pronome/adjetivo flexiona
- "Onde" pode ser advérbio de lugar ou pronome relativo
- "É proibido fumar" → "fumar" = substantivo (infinitivo substantivado)
- Artigo antes de pronome: "O eu" / "O não" → nominalização

## 📌 Adjetivos eruditos e pátrios
Adjetivos eruditos = formas latinas/gregas para substantivos: *Brasil → brasileiro (popular) / brasílico (erudito)*

Adjetivos pátrios comuns:
| País/Cidade | Adjetivo |
|-------------|----------|
| Florianópolis | florianopolitano |
| Minas Gerais | mineiro |
| Santa Catarina | catarinense |
| Pará | paraense |
| Israel | israelense / israelita / israelítico |
| Paris | parisiense |
| Portugal | português / luso (erudito) |
| Roma | romano / romano |

## 🃏 Flashcards
#flashcards

Qual classe gramatical é invariável (não flexiona em gênero nem número)?::Preposição, Conjunção e Interjeição são invariáveis. Advérbio também (exceto grau).

"Muito" — quando é advérbio e quando é pronome/adjetivo?::Advérbio quando modifica verbo, adjetivo ou outro advérbio (invariável). Pronome/adjetivo quando acompanha ou substitui substantivo (flexiona: muita, muitos, muitas).

Qual é o adjetivo pátrio de Florianópolis?::Florianopolitano(a).

"Bastante" na frase "Ele é bastante corajoso" é qual classe?::Advérbio (modifica o adjetivo "corajoso", invariável).
