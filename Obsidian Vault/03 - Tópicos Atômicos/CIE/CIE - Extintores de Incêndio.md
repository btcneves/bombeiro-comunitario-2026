# CIE — Extintores de Incêndio

**Referência**: Manual CIE CBMSC — Lição I, pp. 26-32

## ⚡ Resumo Brutal

### Tipos de Extintores e aplicações

| Extintor | Classes | Método | NÃO usar em |
|---------|---------|--------|-------------|
| **Água (pressurizada)** | A | Resfriamento | B, C (conduz eletricidade) |
| **Água com aditivo** | A, B | Resfriamento + abafamento | C |
| **Espuma mecânica (AFFF)** | A, B | Abafamento + resfriamento | C |
| **Pó ABC (bicarb. sódico/potássio)** | A, B, C | Inibição da reação em cadeia | Equipamentos eletrônicos sensíveis |
| **Pó BC** | B, C | Inibição | A (não extingue brasa) |
| **CO₂ (gás carbônico)** | B, C | Abafamento + resfriamento | A (não extingue brasa), D |
| **Pó especial (Classe D)** | D | Abafamento | Outros |

### Operação do extintor (PASSE)
1. **P**uxar o pino de segurança
2. **A**pontar o bico para a base das chamas
3. **S**pressionar o gatilho
4. **S**arrear (varrer) da base das chamas

### Partes do extintor
- Corpo cilíndrico (reservatório do agente)
- Válvula/manômetro (pressão)
- Pino de segurança
- Mangueira/bico
- Manômetro (verde = carga normal; vermelho = descarregado/sobrecarregado)

### Manutenção e validade
- Inspeção visual mensal
- Recarga/teste hidrostático: conforme ABNT NBR (prazo indicado no corpo)
- Extintor com manômetro no vermelho = inoperante = infração

## 🎯 Pegadinhas
- **Pó ABC** apaga classe C mas **pode danificar equipamentos eletrônicos** — para central de computadores, usar CO₂
- **CO₂** não combate Classe A (não resfria o material sólido suficientemente)
- Em incêndio elétrico: DESLIGAR a energia → passa a ser Classe A ou B → aí pode usar água
- AFFF (espuma) forma película sobre líquidos — proibido usar em elétrico

## 🃏 Flashcards
#flashcards

Qual extintor usar em incêndio elétrico (Classe C)?::CO₂ ou Pó Químico Seco (ABC ou BC). NUNCA água ou espuma.

Qual é a sigla para a técnica de operação do extintor?::PASSE — Puxar o pino, Apontar para a base, Pressionar o gatilho, Varrer da base das chamas.

O CO₂ combate incêndio Classe A?::Não. CO₂ não resfria o material sólido suficientemente para extinguir brasa. Adequado para B e C.

Qual a diferença entre Pó BC e Pó ABC?::Pó BC apaga apenas B e C. Pó ABC apaga A, B e C (mais completo). O ABC contém agente que retarda a combustão de sólidos.
