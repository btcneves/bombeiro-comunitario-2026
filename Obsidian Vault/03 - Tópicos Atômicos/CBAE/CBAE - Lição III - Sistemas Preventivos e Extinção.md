# CBAE — Lição III — Sistemas Preventivos e Extinção de Princípios de Incêndios

**Referência**: Manual CBAE CBMSC — Lição III, pp. 55-83

## ⚡ Resumo Brutal

### Sistemas Preventivos de Incêndio
| Sistema | Função |
|---------|--------|
| **Extintores** | Combate a princípios de incêndio (extinção inicial) |
| **Sistema Hidráulico Preventivo (SHP)** | Rede de mangueiras e hidrantes internos para combate manual |
| **Chuveiros automáticos (sprinklers)** | Ativam automaticamente por calor — controlam/extinguem incêndio |
| **Detecção e alarme** | Detectores de fumaça/calor + sirene → alertar ocupantes |
| **Saídas de emergência** | Rotas de fuga sinalizadas, sempre desobstruídas |
| **Iluminação de emergência** | Ativa automaticamente na falta de energia |
| **Sinalização de abandono** | Placas verdes: "saída", seta de direção, ponto de encontro |

### Plano de Emergência
- Documento que define procedimentos em caso de incêndio/emergência
- Inclui: organograma de brigada, pontos de encontro, número de emergência, rotas de fuga
- **Simulado**: treino periódico de evacuação — obrigatório para brigadas

### Triângulo do Fogo (CBAE usa o triângulo, não o tetraedro — nível básico)
- Combustível + Comburente + Calor → remove um → extinção

### Agentes Extintores (revisão pelo CBAE)
- Água: classe A
- CO₂: classe B e C
- Pó químico ABC: classes A, B, C
- Espuma: classes A e B

### Técnica de extinção de princípio de incêndio com extintor
1. Verificar: classe do incêndio × tipo do extintor
2. Segurar o extintor corretamente
3. Sacar o pino
4. Apontar para a BASE das chamas
5. Pressionar o gatilho
6. Varrer da esquerda para a direita
7. Manter saída desobstruída

## 🎯 Pegadinhas
- Sprinklers NÃO apagam todo o incêndio — apenas CONTROLAM e retardam a propagação
- Saídas de emergência: NUNCA trancar ou obstruir
- Detector de fumaça ≠ detector de calor: fumaça ativa mais rápido (antes das chamas)

## 🃏 Flashcards
#flashcards

O que faz o sistema de chuveiros automáticos (sprinklers)?::Ativam automaticamente ao atingir temperatura definida (fusível). Controlam ou extinguem princípios de incêndio sem necessidade de intervenção humana imediata.

Para qual classe de incêndio NÃO se pode usar extintor de CO₂?::Classe A (sólidos que formam brasa). CO₂ não resfria o material suficientemente para extinguir a brasa residual.

Por que deve-se apontar o extintor para a BASE das chamas e não para o topo?::O agente extintor age sobre o combustível (base). Apontar para as chamas apenas as abafa momentaneamente sem eliminar a fonte.
