# Flashcards

## Padrão dos cards

Os cards usam o formato:

```text
Pergunta::Resposta
```

Esse padrão mantém os decks legíveis no Obsidian e facilita eventual
exportação seletiva para Anki.

## Regra principal

Cada card deve cobrar um conceito por vez. Se a resposta exigir vários passos
ou virar resumo de capítulo, o card deve ser dividido.

## Boas práticas

- Usar pergunta objetiva.
- Escrever resposta curta.
- Cobrar termos usados pelas fontes oficiais.
- Preferir cards derivados de notas atômicas já revisadas.
- Reforçar cards quando houver erro em simulado.

## Evitar

- Cards fora do edital.
- Cards baseados em fonte informal como se fosse fonte principal.
- Respostas longas demais.
- Cards duplicados em decks diferentes.
- Mnemônicos externos sem vínculo claro com o conteúdo oficial.

## Preparação para Anki

Antes de exportar:

1. confirme se o card está correto;
2. confirme se ele tem uma resposta curta;
3. confira se não existe card equivalente no Anki;
4. exporte em lote pequeno;
5. mantenha o Obsidian como fonte principal de correção.
