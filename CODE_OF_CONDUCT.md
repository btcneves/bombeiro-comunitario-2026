# Código de Conduta

## Compromisso

Este projeto adota um padrão de colaboração profissional, respeitoso e objetivo. Espera-se que toda interação em issues, pull requests e discussões mantenha foco técnico e respeito mútuo.

## Comportamentos esperados

- comunicar críticas com clareza e respeito
- assumir boa-fé dos participantes
- priorizar fatos, contexto e reprodutibilidade
- aceitar correções de forma profissional
- contribuir para um ambiente útil a estudantes e colaboradores

## Comportamentos inaceitáveis

- ataques pessoais
- linguagem ofensiva ou discriminatória
- provocação deliberada
- exposição de dados pessoais sem autorização
- sabotagem de conteúdo, documentação ou estrutura do projeto

## Aplicação

Mantenedores podem editar, ocultar ou remover conteúdo incompatível com este código de conduta e limitar participação de usuários que insistam em conduta inadequada.

## Escopo

Este código se aplica a interações no repositório, incluindo issues, pull requests, comentários e propostas de conteúdo.
