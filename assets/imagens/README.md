# Imagens do Projeto

Esta pasta deve receber screenshots reais do projeto quando houver captura
visual validada no Obsidian.

## Imagens recomendadas

- Print do `🎯 Dashboard`.
- Print de uma nota atômica.
- Print de um deck de flashcards.
- Print de um simulado.
- Print da matriz de desempenho.

## Critério

Não adicionar imagens ilustrativas genéricas. As imagens devem mostrar o
estado real do vault e preservar dados sensíveis, se houver.
