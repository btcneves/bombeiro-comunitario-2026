# Bombeiro Comunitário 2026

Sistema profissional de estudo com Obsidian para Bombeiro Comunitário 2026,
fiel ao edital e organizado com notas atômicas, flashcards, revisão espaçada,
simulados, matriz de desempenho e auditoria de fontes.

## Plataforma de estudo

O núcleo do projeto é o `Obsidian Vault`, estruturado para operar a preparação
diária:

- notas atômicas para dividir o conteúdo em unidades revisáveis;
- flashcards para active recall;
- revisão espaçada para retenção;
- simulados e questões para medir domínio;
- matriz de desempenho para acompanhar evolução;
- sistema de erros para transformar falhas em revisão.
