# Metodologia

## Visão geral

O projeto foi construído para substituir estudo passivo por um sistema de retenção ativa. A ideia central é simples: estudar não é consumir páginas, mas consolidar memória útil, recuperar informação sob pressão e corrigir falhas de forma rastreável.

## O que são notas atômicas

Notas atômicas são unidades pequenas de conhecimento. Em vez de resumir um capítulo inteiro em uma nota gigante, cada conceito relevante ganha uma nota específica, com contexto suficiente para ser entendido e revisado sem depender de blocos longos.

Essa granularidade melhora:

- foco cognitivo
- busca e recuperação
- revisão segmentada
- criação de links semânticos
- criação de flashcards melhores

## Por que uma nota por conceito melhora retenção

Quando várias ideias são agrupadas em uma única nota longa, a revisão vira releitura difusa. Já uma nota por conceito força recorte, clareza e decisão sobre o que realmente precisa ser entendido.

Esse modelo:

- reduz ambiguidade
- facilita a identificação do ponto fraco real
- permite revisar apenas o que está vencido
- melhora a ligação entre conceito, erro e card

## Como funciona Zettelkasten leve

O projeto não replica um Zettelkasten acadêmico puro. Ele aplica uma versão leve, pragmática e orientada a concurso:

- notas curtas
- links internos relevantes
- conexões entre assuntos correlatos
- estrutura navegável por disciplina e por uso

O objetivo não é criar uma enciclopédia abstrata, mas uma malha de conhecimento fácil de consultar, revisar e recombinar.

## Como links internos ajudam memória associativa

Memória humana funciona melhor em rede do que em listas isoladas. Quando uma nota sobre `Fundamentos do Fogo` aponta para `Extintores de Incêndio`, `EPI` e `Técnicas de Combate Interno`, ela cria contexto recuperável.

Na prática, os links internos:

- fortalecem associação entre tópicos
- ajudam a detectar lacunas
- aceleram revisão contextual
- reduzem duplicação de explicações

## Como Active Recall evita estudo passivo

Active Recall é o esforço de puxar a resposta da memória antes de reler. É o oposto de “achei familiar, então devo saber”.

Neste sistema, ele aparece em:

- flashcards
- perguntas internas nas notas
- revisão de erros
- consulta ao dashboard para atacar pendências reais

O benefício é direto: o estudante mede recuperação, não sensação de entendimento.

## Como Spaced Repetition reduz esquecimento

O projeto usa revisão espaçada com ciclos curtos e progressivos:

- `D+1`: consolidação imediata
- `D+7`: reforço após a primeira curva de esquecimento
- `D+30`: retenção de médio prazo

Ao distribuir revisão em janelas certas, o sistema evita tanto o abandono quanto a repetição desnecessária.

## Como o sistema de erros transforma falhas em revisão direcionada

Erro sem registro vira esquecimento reincidente. Erro registrado vira insumo operacional.

O fluxo esperado é:

1. errar em questão, revisão ou simulado
2. registrar o erro com causa e correção
3. ligar o erro à nota atômica correspondente
4. criar ou reforçar flashcard
5. reintroduzir o tema no ciclo de revisão

Esse processo muda o papel da falha. Em vez de ser só frustração, ela passa a indicar exatamente onde o sistema precisa reforço.

## Resultado metodológico

O conjunto de notas atômicas, links internos, revisão espaçada, flashcards e rastreamento de erros forma um sistema de aprendizado ativo com baixa fricção e alta rastreabilidade. Ele foi desenhado para estudo sério, repetível e auditável, não para arquivamento passivo.
