# Adaptando para outros concursos

Este projeto foi desenhado para Bombeiro Comunitário 2026, mas a arquitetura
pode ser reutilizada para outros editais sem recomeçar do zero.

## 1. Substituir edital

Atualize a pasta de referências e as notas em `01 - Edital` com o novo
conteúdo oficial. O edital continua sendo a fonte primária de escopo.

## 2. Criar MOCs por disciplina

Em `02 - Disciplinas`, ajuste ou recrie páginas centrais com base nas matérias
do novo concurso.

## 3. Criar notas atômicas

Em `03 - Tópicos Atômicos`, gere novas notas por conceito ou por subtópico do edital. Preserve o padrão de frontmatter.

## 4. Criar flashcards

Monte decks novos em `04 - Flashcards` e adicione cards dentro das notas atômicas quando fizer sentido.

## 5. Ajustar dashboard

Revise consultas Dataview, prioridades e atalhos do dashboard para refletir as novas disciplinas e metas.

## 6. Ajustar cronograma

Atualize plano de estudo, ciclo de revisão, datas e prioridades conforme a prova-alvo.

## 7. Atualizar README

Mude o posicionamento público do projeto para refletir o novo concurso,
mantendo a metodologia e a arquitetura como base reutilizável.

## Regra de adaptação

Adapte conteúdo, não destrua sistema. O valor principal do repositório está na
combinação entre método, estrutura e operacionalização do estudo.
