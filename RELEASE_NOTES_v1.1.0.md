# Bombeiro Comunitário 2026 — v1.1.0

## Visão geral

Versão de consolidação do roadmap atual. O projeto passa a ter documentação
final de maturidade, diagramas, integração Anki documentada, simulados
avançados, GitHub Pages consolidado e processo de adaptação para outros
concursos.

## Destaques

### Roadmap atual consolidado

O roadmap público foi atualizado para refletir o estado real do projeto:
`v1.0.0`, `v1.0.1`, `v1.1`, `v1.2`, `v1.3`, `v1.4` e a documentação de
adaptabilidade `v2.0` estão concluídas ou documentadas.

### Diagramas profissionais

Foram adicionados diagramas Mermaid para:

- arquitetura do vault;
- fluxo de estudo completo;
- ciclo de melhoria;
- fidelidade ao edital.

### Integração com Anki

A integração foi documentada como fluxo seletivo:

```text
Obsidian como fonte -> revisão local -> exportação seletiva -> Anki para repetição intensiva
```

### Simulados avançados

O sistema de simulados ganhou plano operacional, checklist pós-simulado e
documentação para análise por faixa de desempenho.

### GitHub Pages

`docs/index.md` foi consolidado como landing page pública e `docs/_config.yml`
foi adicionado para publicação via Jekyll a partir de `/docs`.

### Adaptação para outros concursos

Foi documentado como reaproveitar a metodologia sem confundir o escopo: este
repositório permanece específico para Bombeiro Comunitário 2026, mas o método
pode ser adaptado com novo edital, fontes, MOCs, notas, flashcards, simulados
e release própria.
